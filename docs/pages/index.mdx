# Reactfire SSR

Reactfire SSR builds on top of Reactfire to provide server-side rendering support for firebase when using reactfire.

## Installation

Required dependencies:

- reactfire-ssr (duh!)
- reactfire
- firebase
- firebase-admin

<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tab>
  ```bash copy
  pnpm install reactfire-ssr reactfire firebase firebase-admin
  ```
  </Tab>
  <Tab>
  ```bash copy
  npm install reactfire-ssr reactfire firebase firebase-admin
  ```
  </Tab>
  <Tab>
  ```bash copy
  yarn add reactfire-ssr reactfire firebase firebase-admin
  ```
  </Tab>
</Tabs>
